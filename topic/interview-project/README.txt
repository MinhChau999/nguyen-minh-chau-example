Yêu cầu:

* Stack

- Backend API: Laravel (nếu không làm về backend thì dùng giả lập JSON API)
- Frontend: NuxtJS (VueJS), Bootstrap (hoặc tự code CSS)

* Chức năng:

- Đăng ký tài khoản
- Đăng nhập
- Liệt kê danh sách người dùng, có phân trang.
- Tạo tài khoản mới (từ Admin Dashboard).
- Sửa thông tin người dùng.
- Xóa người dùng.

* Làm việc bằng Git.
